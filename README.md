## 🛸 Misión Control

**Nocturne Neon** es un portafolio de alto rendimiento diseñado para el vacío del espacio digital. Construido con una arquitectura de **Soberanía Profesional**, este sistema elimina el peso muerto de los frameworks para alcanzar velocidades de carga hiperlumínicas.

### 🔭 Objetivos de la Sonda:

- **Rendimiento Implacable:** Optimizado para Core Web Vitals (LCP < 1.2s).
- **Encapsulamiento:** Uso de Web Components nativos para una estructura modular.
- **Bilingüe:** Sistema de navegación EN/ES mediante diccionarios JSON.

---

## 🛠️ El Transbordador (Stack Técnico)

| Sistema       | Tecnología   | Función                                    |
| :------------ | :----------- | :----------------------------------------- |
| **Núcleo**    | `Vanilla JS` | Control total sin dependencias externas.   |
| **Propulsor** | `Vite`       | Compilación y HMR ultrarrápido.            |
| **Blindaje**  | `SASS`       | Estética Nocturne Neon basada en tokens.   |
| **Antena**    | `Bunny.net`  | DNS, CDN y WAF para entrega global.        |
| **Hangar**    | `Netlify`    | Hosting estático y gestión de formularios. |
